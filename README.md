# artifuse-voice-conversion

Personal public backup repository for this voice conversion project.

Included:
- source code
- configs and experiment plans
- staged evaluation outputs and reports
- intermediate model/checkpoint artifacts that are safe to publish

Not included:
- raw/source audio
- extracted game assets
- local runtime binaries and temporary files

This repository is maintained as a working archive, not as polished public documentation.
